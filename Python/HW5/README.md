# Advert Class Implementation

This implementation provides the `Advert` class as specified in the homework assignment.

## Features

- Dynamic attribute creation from JSON objects
- Nested attribute access through dot notation
- Automatic handling of Python keywords
- Price validation (non-negative values)
- Default price of 0 when not specified
- Required title field
- Colorized console output

## Usage

The implementation consists of a single file `advert.py` containing:
1. `ColorizeMixin` - Provides colored console output
2. `JSONObject` - Helper class for nested attribute access
3. `Advert` - Main class implementing all required functionality

The class can be used as specified in the assignment requirements.
